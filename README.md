# 🐞 Debugging Assistant App

<p align="center">
A modern Flutter debugging assistant designed to help developers track errors, store solutions, and organize debugging knowledge with a clean production-ready UI.
</p>

---

## ✨ Key Features

* Add debugging errors with solutions
* Edit and delete entries
* Favorite important errors ⭐
* Smart search functionality
* Favorites highlight strip
* Dark / Light mode
* Smooth UI animations
* Glassmorphism design
* Responsive layout
* Real developer workflow simulation
